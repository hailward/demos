* "SOUGOU:3857" <=> "BAIDU:3857"

## 二、工具介绍

1. 使用流程

* 点击上传文件按钮，上传要进行转换的文件，目前仅支持使用 excel 文件。
* 在弹出的对话框中选择数据所在的 sheet 和经纬度对应的字段。
* 配置源坐标和目标坐标。
* 点击导出按钮即可导出转换后的 excel 文件。

2. 注意事项

* excel 文件内容须遵守一定的格式，可点击下载模板文件以便进行参考，格式错误将导致转换失败。
* 单个 excel 文件行数请控制在一千以内，否则可能会导致文件读取失败。
